# 📈 Multi-Stock Price Prediction using LSTM & GRU

A deep learning project for predicting stock prices using LSTM and GRU models on multi-stock time series data.

## 🚀 Features
- 📊 Multi-stock support (AAPL, MSFT, GOOG, etc.)
- 🔄 Time series sequence generation (sliding window)
- 🧠 LSTM & GRU model comparison
- 📉 Next-day and multi-step prediction
- ⚙️ Hyperparameter tuning support
- 📈 Visualization of predictions vs actual values

## 🛠️ Tech Stack
- Python
- PyTorch
- Pandas, NumPy
- Matplotlib
- yFinance API
